To run 2 class classification:

Run:
Bayes_2class.py
KNN_2class.py


To run N class classification:
Run:
Bayes.py
KNN.py

After running the files select the dataset by entering the corresponding choice number. 
For example: for data.mat enter 1. 

Next choose between original dataset or reduced dataset based on PCA or LDA in the same way as above. 


Make sure the datasets are stored in the same folder as the one containing the scripts.
